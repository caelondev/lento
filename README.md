# Lento
A tree-walk interpreter made in Go language

### Getting started
Get started with Lento, a simple and efficient tree-walk interpreter built with Go.

#### Installation
To start writing code in Lento, follow these steps:

#### Prerequisites
- Go version >= 1.25.4 installed on your system

#### Install Lento
Run the following command in your terminal:
```bash
go install github.com/caelondev/lento
```
